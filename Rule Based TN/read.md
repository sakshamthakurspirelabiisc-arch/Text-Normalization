## Rule-Based Text Normalization Models

This folder contains the rule-based text normalization (TN) systems for Hindi and Kannada.

### Hindi

* File: `TB_RB.py`
* Function: `normalize_sent()`

Pass an unnormalized Hindi sentence to the `normalize_sent()` function to obtain its normalized spoken-form representation.

### Kannada

* File: `RB_kan.py`
* Function: `normalize_sent_kan()`

Pass an unnormalized Kannada sentence to the `normalize_sent_kan()` function to obtain its normalized spoken-form representation.

These rule-based systems can be used as baselines for comparison with the neural text normalization models provided in this repository.
